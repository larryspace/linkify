# Linkify
Supposed to be a reddit clone

## Setup
1. `composer install`
1. `npm install`
1. Copy `.env.example` to `.env` and edit it with your environment config.
1. `npm start`

## Build Production
1. `npm run dist`

## Requirements
* [Composer](https://getcomposer.org/)
* [Node.js](https://nodejs.org/)
